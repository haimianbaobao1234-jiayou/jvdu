# 需求文档 — 句读

## 用户画像
- 不懂代码的普通用户
- 英语学习者，需要阅读原汁原味英文文章
- 使用 iPhone 手机
- 预算为零，不接受任何费用

## 核心功能清单

### P0（必须有，否则不可用）
| 功能 | 描述 |
|------|------|
| 文章浏览 | 首页展示文章列表，按难度分类（四级/六级/雅思） |
| 分类筛选 | 顶部标签切换：四级 / 六级 / 雅思 / 全部 |
| 文章阅读 | 点击文章进入阅读页，显示完整英文内容 |
| 句子翻译 | 点击英文句子 → 展开该句中文翻译；再次点击收起 |
| 单词查询 | 点击单词 → 弹出词典弹窗（音标 + 释义 + 发音） |
| 主题切换 | 三种主题色可选：浅绿 / 淡蓝 / 淡粉 |
| PWA 安装 | iPhone Safari 添加到主屏幕，全屏体验 |

### P1（重要，但可后续迭代）
| 功能 | 描述 |
|------|------|
| 自动更新 | 每 2-3 天自动更新文章内容 |
| 阅读记录 | 记录已读文章，显示"已读"标记 |
| 收藏文章 | 收藏喜欢的文章，单独查看 |

### P2（锦上添花）
| 功能 | 描述 |
|------|------|
| 搜索文章 | 按关键词搜索文章标题和内容 |
| 字体大小调整 | 用户可调字号（大/中/小） |
| 夜间模式 | 深色背景适合夜间阅读 |

## 非功能需求
- **费用**：零费用，所有服务必须免费
- **平台**：iPhone Safari（iOS 15+）
- **加载速度**：首屏 3 秒内
- **数据量**：每次更新 3-5 篇新文章，保留近 60 天
- **无广告**：干净阅读体验

## 验收标准
1. iPhone Safari 打开网页 → 看到文章列表
2. 切换分类标签 → 列表正确过滤
3. 点击文章 → 进入阅读页，显示英文
4. 点击英文句子 → 中文翻译出现，再次点击消失
5. 点击单词 → 弹出词典，显示音标和释义
6. 点击主题切换按钮 → 在三种颜色间循环切换
7. 添加到主屏幕 → 全屏打开，有图标
