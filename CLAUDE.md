# 句读 — 英语阅读 PWA 应用

## 项目简介
「句读」是一个运行在 iPhone 上的 PWA 英语阅读应用。
- 自动从 RSS 抓取英文文章，按四级/六级/雅思分类
- 点击句子显示中文翻译
- 点击单词显示释义和发音
- 三种主题色可切换（浅绿/淡蓝/淡粉）
- 零费用运行

## 重要：标准文件路径

在每次工作开始前，必须阅读以下标准文件：
- 📋 需求文档：`docs/01-requirements.md`
- 🔧 技术规范：`docs/02-tech-spec.md`
- 🎨 设计规范：`docs/03-design-spec.md`
- 📝 执行计划：`docs/04-execution-plan.md`

## 重要：开发日志规则

每次工作结束后，必须在 `dev-logs/` 下写入当日日志文件（格式为 `YYYY-MM-DD.md`）：
- 记录今日完成事项
- 记录今日遇到的坑和解决方案
- 记录当前待办事项
- 参考模板：`dev-logs/template.md`

## 工作原则

1. **小步推进**：每次只执行一个步骤，验证通过后再进行下一步
2. **每步验证**：每个步骤结束必须有检查点
3. **记录优先**：每次工作前先读 dev-logs 了解进度；每次工作后写入日志
4. **先读文档**：写代码前先读 docs/ 中的对应规范

## 技术栈

- React 18 + TypeScript
- Vite 构建
- PWA（manifest + Service Worker）
- 纯 CSS（无 UI 框架）
- GitHub Pages 部署
- GitHub Actions 自动化

## 禁止事项

- ❌ 不安装收费依赖
- ❌ 不使用需要 API Key 的服务
- ❌ 不引入 UI 组件库（保持简洁）
- ❌ 不过度设计（这是一个个人小应用）
